Help

Everything you need to know about using the Customer Segmentation Tool

How to Use the Tool

Log In / Sign Up

To start using the tool, log in with your account or sign up if you are a new user.

Upload Sales Data

Click on the "Upload Sales Data" button, select a properly formatted CSV file, and upload it to the tool.

View Results

The segmentation results will display high-selling, mid-selling, and low-selling products with graphical summaries like bar charts and pie charts.

Manage Stock Information

You can add or update stock information using the "Add Stock" button. The stock table will automatically update based on uploaded sales data.

Access Profile History

View your recent results or all previous processed results through your profile section.

Common Issues and Solutions

Issue: CSV File Not Uploading

Make sure the file format is CSV, and ensure it follows the required structure: columns for product names, sales, etc.

Issue: Incorrect Results

If the segmentation results are not as expected, verify the accuracy of your data and ensure the file is not corrupted.

Issue: Stock Updates Not Reflecting

Ensure you have uploaded the correct sales data. The tool will update the stock automatically based on the sales data provided.
